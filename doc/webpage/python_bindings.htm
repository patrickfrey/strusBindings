Python language bindings

Introduction

This chapter describes the language bindings of strus for Python with examples of how to create and use them.
The language bindings have been built using boost.python based on the interface (strus/bindingObjects.hpp).

Overview

For using strus with python you have to load the module strus. Currently there exists no documentation for the python interface. Please refer to the description of the PHP interface here. The interface is roughly the same. The method calls are equal. Only the class names in PHP have the prefix Strus, whereas in Python we have the namespace strus. So we do not need to prefix the data type names to avoid name clashes.

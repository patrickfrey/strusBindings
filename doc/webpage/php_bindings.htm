PHP language bindings

Introduction

This chapter describes the language bindings of strus for PHP with examples of how to create and use them.
The PHP language bindings have been built using SWIG based on a wrapper C++ interface.

Doxygen interface documentation

There exists a documentation for the PHP interface generated with doxygen. You will also find there some examples to introduce the PHP bindings.
